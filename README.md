## 🚀 How to Use

1. **Open any module** by double-clicking the `.html` file — it runs entirely in your browser.
2. **Read the introduction** to learn key concepts and strategies.
3. **Take the quiz** to test your understanding with instant feedback.
4. **Track your progress** — each quiz gives you a score and tips for improvement.

> 💡 **Tip:** These files work offline. Save the folder to your laptop or USB drive and study anywhere.
