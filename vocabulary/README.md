# Math & Vocabulary Learning Modules

A self-contained, interactive learning suite for students to master **Math (Decimals & Mixed Numbers)** and **Vocabulary (in Context)**. No installation, no internet, no accounts — just open and learn.

---

## 🚀 Math Module Setup

1. **Download the `math` folder** as a `.zip` file.
2. **Extract** the `.zip` file to your computer.
3. **Open `math/intro.html`** by double-clicking it. It will open in your web browser.
4. Read the introduction, then click **\"Start the Quiz\"** to begin practicing decimals and mixed numbers.

---

## 🚀 Vocabulary Module Setup

1. **Download the `vocabulary` folder** as a `.zip` file.
2. **Extract** the `.zip` file to your computer.
3. **Open `vocabulary/intro.html`** by double-clicking it. It will open in your web browser.
4. Read the introduction, then click **\"Start the Quiz\"** to begin vocabulary practice.

---

> **Tip:** Both modules work entirely offline. Save folders for easy access.

---

## 🎓 5 Tips for Students (Math & Vocabulary)

1. **Practice daily.** Spend 10-15 minutes — consistency beats cramming.
2. **Read carefully.** Intro examples in both modules prepare you for the quiz.
3. **Use feedback.** Learn from explanations on wrong answers.
4. **Apply skills.** Use decimals in real life; practice vocab in sentences.
5. **Take breaks.** Return refreshed for better results.

---

## 🍎 5 Tips for Teachers & Parents

1. **Start with intros.** `intro.html` teaches key concepts first.
2. **Fully offline.** Ideal for any environment.
3. **Self-paced.** Instant feedback reduces supervision.
4. **Alternate modules.** Balance Math and Vocabulary practice.
5. **Discuss feedback.** Deepen understanding together.
