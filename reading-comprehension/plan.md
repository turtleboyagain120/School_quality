# Reading Comprehension Resource — Implementation Plan

## Overview
Build two polished, interactive HTML pages that teach and test reading comprehension.

## Information Gathered
- No existing comprehension/quiz content in the workspace.
- Will create everything from scratch in a new folder.

## Plan
1. Create a new directory: `reading-comprehension/`
2. Build `intro.html`:
   - Elegant, modern design with warm academic styling.
   - Explain what reading comprehension is.
   - Include a short example passage.
   - Demo a sample explicit & implicit question.
   - Add a button to proceed to the quiz page.
3. Build `quiz.html`:
   - Same visual theme as the intro for consistency.
   - Provide a single, rich reading passage (~200–250 words).
   - Present 10 varied questions (main idea, detail, inference, vocabulary, tone, etc.).
   - Provide instant feedback per question (correct/incorrect).
   - Track/show score at the end.
   - Make it fully self-contained (CSS embedded, no external dependencies).

## Dependent Files
- `intro.html`
- `quiz.html`

## Follow-up Steps
- Open `intro.html` in browser to verify styling and interactions.
